☕ Coffee Sales Analysis Dashboard

📌 Project Overview

This project is a data analysis dashboard created using Microsoft Excel.
It provides insights into coffee sales data including trends, product performance, location-based sales, and customer buying patterns.

---

🎯 Objectives

- Analyze overall coffee sales performance
- Identify top-selling products
- Understand monthly and hourly sales trends
- Compare sales across different store locations
- Visualize data using interactive dashboards

---

📊 Dashboard Features

- Total Quantity Sold
- Total Transactions
- Total Revenue
- Monthly Sales Trend (Line Chart)
- Location-wise Sales (Donut Chart)
- Hourly Sales Analysis (Line Chart)
- Top Selling Products (Bar Chart)
- Top 10 Products by Quantity
- Sales by Product Category

---

🎛️ Filters (Slicers)

The dashboard includes interactive filters for:

- Month Name (January, February, March, etc.)
- Product Category (Coffee, Tea, Bakery, etc.)
- Store Location (Astoria, Hell's Kitchen, Lower Manhattan)

---

🛠️ Tools & Technologies Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning & Formatting

---

📈 Key Insights

- Coffee is the highest selling category
- Peak sales hours are between morning and early afternoon
- Certain locations generate higher revenue than others
- A few top products contribute significantly to total sales

---

📂 Dataset Details

The dataset contains the following columns:

- Transaction ID
- Transaction Date
- Transaction Time
- Transaction City
- Store ID
- Product ID
- Unit Price
- Product Category
- Product Type
- Product Detail
- Size
- Month Name
- Day Name
- Hours
- Sales

---

🚀 How to Use

- Open the Excel file
- Go to the Dashboard sheet
- Use slicers to filter data
- Analyze charts and insights dynamically

---

📌 Conclusion

This dashboard helps in making data-driven decisions by providing clear insights into sales performance, customer behavior, and product demand.
